# Images

All photographs and SDK screenshots in this directory document GB9110 development.

- `continuous-loop.jpg`: continuous ROM execution in the SDK environment;
- `first-real-frame.jpg`: first real frame diagnostic;
- `hardware-profile-summary.jpg`: original cumulative profiler summary;
- `hardware-profile-opcodes.jpg`: original opcode histogram;
- `profiler-opcode-core.jpg`: current rate-based opcode profile, GBPROF v1.5j;
- `profiler-memory-core.jpg`: current data-memory profile with fetch bytes excluded;
- `profiler-both-fs2.jpg`: combined opcode/memory profiler under FS2;
- `div2-core-v2.jpg`: GBDIV2 v1.6 CPU-only DIV v2 result;
- `div2-fs2-v2.jpg`: GBDIV2 v1.6 FS2 DIV v2 result.

The hardware photographs were taken by the GB9110 project author.
