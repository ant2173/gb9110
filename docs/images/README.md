# Images

All photographs and SDK screenshots in this directory document GB9110 development.

- `continuous-loop.jpg`: continuous ROM execution in the SDK environment;
- `first-real-frame.jpg`: first real frame diagnostic;
- `hardware-profile-summary.jpg`: profiler summary on a real Nokia 9110;
- `hardware-profile-opcodes.jpg`: opcode histogram on a real Nokia 9110.

The hardware photographs were taken by the GB9110 project author.
