# Project pitch

GB9110 is an AI-assisted, hardware-verified experiment in running a Game Boy emulator on the Nokia 9110 Communicator with the original PC/GEOS SDK and Borland C++ 4.52. A real ROM loads, executes, renders, and accepts input on actual hardware. Profiler-guided CPU and renderer changes have raised the complete path from 5 to 13 FPS, including a device-side opcode profiler, a guarded native superinstruction, and an eight-pixel scanline renderer. The repository documents successful optimizations and instructive failures rather than presenting an unfinished experiment as a polished emulator.
