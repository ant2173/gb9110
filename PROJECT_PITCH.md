# Project pitch

GB9110 is an AI-assisted, hardware-verified experiment in running a Game Boy emulator on the Nokia 9110 Communicator with the original PC/GEOS SDK and Borland C++ 4.52. A real ROM loads, executes, renders and accepts input on actual hardware. Profiler-guided CPU and renderer work raised the complete render-every-frame path from 5 to 13 FPS; fixed FS2 now advances the guest at 18 FPS while displaying 9 FPS, and the CPU-only path reaches 28 FPS. The repository includes device-side opcode and memory profilers, guarded ROM-specific division helpers, an eight-pixel scanline renderer, successful frame skipping and documented failures. The next milestone is a narrow same-binary 8086 assembly experiment, not a speculative full CPU rewrite.
