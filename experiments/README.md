# Experiments

The repository keeps selected reproducible milestones rather than every temporary build.

- `gbtable`: first decisive renderer improvement;
- `gbrow`: current best combined CPU/renderer benchmark.

Intermediate builds — GBHOT, GBTIME, GBFLAGS, GBDIV, and the failed GBTILE cache — are documented in [`docs/OPTIMIZATION_HISTORY.md`](../docs/OPTIMIZATION_HISTORY.md). Their final ideas are incorporated into `gbrow` where applicable.
