# GB9110 опубликован на GitHub

Я публикую GB9110 как открытый инженерный журнал: экспериментальный эмулятор Game Boy для Nokia 9110 Communicator, собранный с помощью оригинального PC/GEOS SDK и Borland C++ 4.52.

Настоящий homebrew-ROM уже загружается, выполняется, рисуется и принимает управление на реальном аппарате. Первый измеренный полный путь кадра работал примерно на 5 FPS. После профилирования прямо на Nokia и серии экспериментов с CPU и рендерером лучший путь с выводом каждого кадра достиг 13 FPS, фиксированный FS2 — 18 гостевых / 9 экранных FPS, а CPU-only путь — 28 FPS.

В репозитории есть:

- ранняя аппаратная игровая сборка;
- профилировщик CPU / PPU / упаковки / blit;
- профилировщик opcode и обращений к памяти на самом устройстве;
- первый успешный табличный рендерер;
- benchmark-сборки ROW8, фиксированного frameskip и layered DIV v2;
- инструкция по исторической среде разработки;
- измеренные неудачи, включая кэш тайлов с почти 100% попаданий, который всё равно замедлил эмулятор.

Проект разрабатывается с помощью ИИ, но каждое утверждение о скорости подтверждается реальной Nokia. Оптимизация не считается успешной только потому, что красиво выглядит в исходнике.

Это не готовый релиз эмулятора. Пока нет звука, Save RAM, выбора ROM и заявлений о широкой совместимости. Фиксированный frameskip и профилирование текущей нагрузки уже завершены; следующий практический шаг — узкий A/B эксперимент на 16-битном x86 ассемблере.

Интересно не только то, что ROM Game Boy запускается на Nokia 9110. Интересно, что аппарат достаточно старый и ограниченный, чтобы сразу показывать цену каждого архитектурного решения.
