# Черновик первого поста о GitHub

Я начал публиковать **GB9110** — экспериментальный эмулятор Game Boy для Nokia 9110 Communicator.

Настоящий DMG-ROM уже:

- загружается из файловой системы коммуникатора;
- выполняется адаптированным ядром Peanut-GB;
- выводится в bitmap GEOS;
- принимает управление с клавиатуры;
- запускается на реальной Nokia 9110.

Полной скорости пока нет, и репозиторий этого не скрывает.

Самая интересная часть — измерения на аппарате:

```text
только CPU:                 21–22 FPS
оригинальный PPU:                7 FPS
PPU и packed-преобразование:     6 FPS
полный кадр с выводом GEOS:      5 FPS
статический blit GEOS 4-bpp:    44 FPS
```

Теперь работа сосредоточена на снижении стоимости PPU, а затем на оптимизации интерпретатора LR35902 для 16-битного x86 и Borland C++ 4.52.

Репозиторий задуман как открытый инженерный журнал: рабочая база, аппаратный профилировщик, текущий экспериментальный рендерер, инструкции по сборке, измерения и отрицательные результаты.

ROM и проприетарные файлы SDK в него не входят.
